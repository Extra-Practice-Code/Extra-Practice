Computer Dance (exquisite coding / against the clockuter) *

A warm up coding practice that occurred on May 17, 2022 at XP.

* https://nl.wikipedia.org/wiki/Stoelendans

Ingredients

  • floor (big table optional)
  • 3+ friends
  • 3+ laptops installed with code editors

Instructions

  1. everyone sits around the big table or on the floor.
  2. start the timer.
  3. when the countdown ends the timer will prompt you to switch computers, continue coding, or take a break.
  4. go for a few rounds and when the it feels right share the websites you've created with each other.
